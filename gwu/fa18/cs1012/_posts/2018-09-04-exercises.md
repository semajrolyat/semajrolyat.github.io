## Paths

## Navigating in the Console
* ```ls``` lists the contents of the current directory
* ```.``` refers to the current directory
* ```..``` referes to the parent directory
* ```cd``` changes the current directory

```
$ ls
```

```
$ ls .
```

```
$ ls ..
```

```
$ cd
```
